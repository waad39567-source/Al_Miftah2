POST /auth/register عام

📝 الوصف

تسجيل مستخدم جديد في النظام. يتم إنشاء حساب غير موثق في البداية.

📋 Headers

المفتاحالقيمة
Content-Typeapplication/json
Acceptapplication/json

📦 Request Body

الحقلالنوعمطلوبالوصفالقواعدمثال
name string نعم الاسم الكامل required, max:255 "اسم المستخدم"
email email نعم البريد الإلكتروني required, email, unique "user@test.com"
password string نعم كلمة المرور required, min:8, confirmed "password123"
password_confirmation string نعم تأكيد كلمة المرور required "password123"
phone string نعم رقم الهاتف required, unique, max:20 "966501234567"
role string لا دور المستخدم in:user,owner "user"

✅ استجابات ناجحة

201 Created
{ "success": true, "message": "تم التسجيل بنجاح...", "data": { "user": { "id": 1, "name": "اسم المستخدم", "email": "user@test.com", "phone": "966501234567", "role": "user", "is_active": true, "created_at": "2026-03-08T12:00:00" } } }

❌ استجابات خاطئة

422 Unprocessable

خطأ في التحقق من البيانات

{ "success": false, "message": "Validation error", "errors": { "email": ["البريد الإلكتروني مستخدم من قبل"], "phone": ["رقم الهاتف مستخدم من قبل"] } }
POST /auth/login عام

📝 الوصف

تسجيل الدخول للحصول على token المصادقة.

📋 Request Body

الحقلالنوعمطلوبمثال
emailstringنعم"user@test.com"
passwordstringنعم"password123"

✅ 200 - نجاح

{ "success": true, "message": "تم تسجيل الدخول بنجاح", "data": { "user": {...}, "token": "1|abc123..." } }

❌ 401 - بيانات خاطئة

{"success": false, "message": "بيانات الاعتماد غير صحيحة"}

❌ 403 - غير موثق

{"success": false, "message": "لم يتم توثيق حسابك بعد..."}

❌ 403 - غير نشط

{"success": false, "message": "الحساب غير نشط"}
POST /auth/send-verification-email عام

⚠️ ملاحظة مهمة

هذا الـ API يضيف البريد الإلكتروني إلى Queue (قائمة الانتظار). يجب تشغيل php artisan queue:work لإرسال البريد فعلياً. رابط التوثيق يُرسل للمستخدم عبر البريد الإلكتروني وليس الواجهة.

📋 Request Body

الحقلالنوعمطلوبمثال
emailstringنعم"user@test.com"

✅ 200 - نجاح

{ "success": true, "message": "تم إضافة إرسال رابط توثيق البريد الإلكتروني إلى قائمة الانتظار", "data": null }

❌ 400 - موثق مسبقاً

{"success": false, "message": "تم توثيق البريد الإلكتروني مسبقاً"}

📝 الوصف

📋 Query Parameters (اختياري)

المعاملالوصفالقيم الممكنةمثال

parent_idفلترة حسب المنطقة الأبرقم أو null?parent_id=1
searchالبحث في الاسمنص?search=رياض
has_childrenفلترة حسب وجود أبناءtrue, false?has_children=true
sort_byحقل الترتيبname, created_at, type?sort_by=name
sort_orderاتجاه الترتيبasc, desc?sort_order=asc

{ "success": true, "data": { "current_page": 1, "data": [ { "id": 1, "name": "السعودية", "type": "country", "parent_id": null, "created_at": "2026-03-08T12:00:00" } ], "total": 10 } }
