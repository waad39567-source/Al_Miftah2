🔄 Cascade Delete

عند حذف المستخدم، تحذف تلقائياً:

  • العقارات المرتبطة بالمستخدم
  • صور العقارات من التخزين
  • طلبات التواصل
  • الإشعارات
  • رموز FCM
  • العقارات المفضلة
  • رموز المصادقة (Tokens)

👤 صلاحيات المستخدم العادي
