📝 الوصف

📋 Headers

المفتاحالقيمة
Content-Typeapplication/json
Acceptapplication/json

📦 Request Body

الحقل النوع مطلوب الوصف القواعد مثال
name string نعم الاسم الكامل للمستخدم required, max:255 "أحمد محمد"
email string نعم البريد الإلكتروني (يجب أن يكون فريد) required, email, unique "ahmed@test.com"
password string نعم كلمة المرور required, min:8, confirmed "password123"
password_confirmation string نعم تأكيد كلمة المرور required, same:password "password123"
phone string نعم رقم الهاتف (يجب أن يكون فريد) required, unique, max:20 "966501234567"
role string لا دور المستخدم in:user,owner "user"

✅ استجابة ناجحة

201 Created
{ "success": true, "message": "تم التسجيل بنجاح. لم يتم توثيق حسابك بعد يرجى التحقق من بريدك الإلكتروني لتوثيق الحساب", "data": { "user": { "id": 1, "name": "أحمد محمد", "email": "ahmed@test.com", "phone": "966501234567", "role": "user", "is_active": true, "created_at": "2026-03-09T10:30:00" } } }

❌ استجابات خاطئة

422 Unprocessable

1. بريد إلكتروني مكرر:

{ "success": false, "message": "Validation error", "errors": { "email": ["البريد الإلكتروني مستخدم من قبل"] } }

2. رقم هاتف مكرر:

{ "success": false, "message": "Validation error", "errors": { "phone": ["رقم الهاتف مستخدم من قبل"] } }

3. كلمة مرور غير متطابقة:
