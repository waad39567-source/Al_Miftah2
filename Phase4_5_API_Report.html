📝 ملفات معدلة - المرحلة 4

AccountService.php معدّل
app/Services/AccountService.php
// دالة جديدة: updateEmail public function updateEmail(User $user, string $email, string $password): bool { // 1. التحقق من كلمة المرور if (!Hash::check($password, $user->password)) { return false; } // 2. التحقق من عدم وجود البريد if (User::where('email', $email)->where('id', '!=', $user->id)->exists()) { return false; } // 3. تحديث البريد $user->update(['email' => $email]); return true; }
AccountController.php معدّل
app/Http/Controllers/AccountController.php
// إضافة Import use App\Http\Requests\UpdateEmailRequest; // دالة جديدة public function updateEmail(UpdateEmailRequest $request): JsonResponse { $user = $request->user(); $result = $this->accountService->updateEmail( $user, $request->validated('email'), $request->validated('password') ); if (!$result) { return $this->errorResponse('كلمة المرور غير صحيحة...', 401); } return $this->successResponse(null, 'تم تغيير البريد الإلكتروني بنجاح'); }
api.php Route جديد
routes/api.php

📋 ملخص الـ APIs

APIs تسجيل الدخول المتاحة الآن:

Method Endpoint Auth الوظيفة
POST /auth/register لا تسجيل مستخدم جديد
POST /auth/login لا تسجيل الدخول (email أو phone)
PUT /auth/profile نعم تعديل الملف الشخصي
PUT /auth/email نعم تغيير البريد الإلكتروني
POST /auth/change-password نعم تغيير كلمة المرور
DELETE /auth/account نعم حذف الحساب
